Algorithm Visualizer

A better visualization of different algorithms made with React.

There are 5 segments

-   Pathfinder
-   Sorting Algorithms
-   N Queen
-   Binary Search Game
-   Binary Tree Traversals

## Algorithms implemented

-   Graph Search
    -   DFS
    -   BFS
      
-   Sorting
    -   Bubble sort
    -   Selection sort
    -   Insertion sort
    -   Merge sort
    -   Quick sort
  
-   N Queen Backtracking
-   Binary Search
-   Binary Tree Traversals
    -   Inorder Traversal
    -   Preorder Traversal
    -   Postorder Traversal
